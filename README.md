# BotanikData

BotanikData ist eine interaktive Anwendung zur Verwaltung von Pflanzenpflege-Daten. Das Tool ermöglicht es, verschiedene Pflanzenarten systematisch zu erfassen und deren spezifische Bedürfnisse übersichtlich darzustellen.

## Kernfunktionen
* **Datenerfassung:** Speicherung von Pflanzen als strukturierte JavaScript-Objekte (beinhaltet Name, Standort und Pflege-Intervalle).
* **Dynamische Anzeige:** Automatische Aktualisierung der Listenansicht bei neuen Einträgen über DOM-Manipulation.
* **Filterung & Organisation:** Übersichtliche Darstellung der Bestände für eine effiziente Alltagsplanung.

## Technische Umsetzung
Das Projekt wurde mit Fokus auf ein modulares Grundgerüst entwickelt:
* **HTML5 & CSS3:** Struktur und responsives Design. Status: Updated 16.01
* **JavaScript (ES6):** Logik zur Datenverarbeitung und dynamischen Elementerzeugung.
* **State-Management:** Trennung von Datenmodell (Array/Objekte) und der visuellen Darstellung (Render-Funktion).

## Roadmap
Zukünftige Erweiterungen für dieses Projekt:
* Implementierung von LocalStorage zur dauerhaften Datenspeicherung.
* Suchfunktion für große Bestände.
* Benachrichtigungs-Logik für anstehende Pflegeaufgaben.
